Haptic diagnostics

1. Real switch (tap it yourself)

If tapping this native switch gives NO haptic, the problem is iOS version (needs 17.4+) or Settings > Sounds & Haptics > System Haptics is off.

2. Programmatic switch click

Same trick SleepNoise uses: JS creates a hidden switch and clicks it.

3. Vibration API

navigator.vibrate - Android only, expected to fail on iPhone.

4. Invisible switch overlay (tap the button)

A real switch covers this button with opacity 0: your tap lands on the actual switch. If this vibrates, the whole UI can work this way.
